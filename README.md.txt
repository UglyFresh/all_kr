Программа разработана для выполнения задания по дисциплине Информационные технологии и программирование
Текст задания:
Необходимо разработать класс для указанной 
предметной области.
Предметная область:
Файл: имя файла, размер, дата создания, количество обращений. Создать 
массив объектов. Реализовать возможность получения:
– списка файлов, упорядоченного в алфавитном порядке,
– списка файлов, размер которых превышает заданный,
– списка файлов, число обращений к которым превышает заданное.
